## Dependencies

- Node.js
- Express
- EJS
- bcryptjs
- cookie-session

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run by using `npm start`
- You can test files by using `npm test`

- First you need to register then you're good to go :)
